# 팀 대시보드 프로젝트

## 프로젝트 소개
이 프로젝트는 팀의 일정, 뉴스, 리스크 이슈 등을 한눈에 볼 수 있는 대시보드 시스템입니다.

### 주요 기능
- 일정 관리 및 공지
- 뉴스 모니터링 (제휴처, 리스크이슈, 신기술동향)
- **이용통계 대시보드 (명령어/액션별, 기간별, 차트/표 시각화, 엑셀 다운로드)**
- **DB 사용량 관리 (제한/삭제량 설정, 자동 용량 초과시 과거 데이터 삭제)**
- **통계 메일링 (월 1회, 관리자 지정 메일로 월간 이용통계 자동 발송)**
- 관리자 페이지
- 카카오톡 봇 연동

### 기술 스택
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express
- Database: MongoDB
- Deployment: Vercel, Render

## 설치 및 실행 방법

### 필수 요구사항
- Node.js 14.x 이상
- MongoDB

### 설치
```bash
# 의존성 설치
npm install

# 환경 변수 설정
# .env 파일 생성 및 필요한 환경 변수 설정
```

### 실행
```bash
# 개발 서버 실행
npm run dev

# 프로덕션 빌드
npm run build
```

## 주요 기능 설명

### 뉴스 모니터링
- 제휴처 뉴스: 파트너사 관련 뉴스 모니터링
- 리스크이슈: 산업 전반의 리스크 관련 뉴스
- 신기술동향: 최신 기술 트렌드 뉴스

### 관리자 기능
- 뉴스 데이터 관리
- 일정 관리
- 시스템 설정
- **이용통계 대시보드**: 명령어/액션별, 일/월/누적 통계, 차트/표, 엑셀 다운로드
- **DB 사용량 관리**: 현재 사용량, 제한/삭제량 설정, 자동 데이터 삭제
- **통계 메일링**: 월 1회, 지정 메일로 월간 이용통계 자동 발송

### 카카오톡 봇
- 일정 조회
- 뉴스 요약
- 알림 설정
- **명령어/액션별 이용통계 자동 기록**

### 일정알림 메일
- 일정 등록/변경/취소 시 메일 발송
- **메일 하단에 대시보드 접속 링크 포함**

## 배포 정보

### 환경 변수
- `MONGODB_URI`: MongoDB 연결 문자열
- `KAKAO_BOT_TOKEN`: 카카오톡 봇 토큰
- `ADMIN_PASSWORD`: 관리자 페이지 비밀번호
- `EMAIL_USER`: 통계/일정 메일 발송용 Gmail 주소
- `EMAIL_PASS`: Gmail 앱 비밀번호

### 배포 환경
- Frontend: Vercel
- Backend: Render
